# TV Box Linux — RK322x

Documentação do processo de descaracterização de TV Box com chip RK322x,
instalação do Debian, ativação do WiFi (SSV6501P) e configuração de Access Point.

## Hardware
- SoC: Rockchip RK3228A (RK322x)
- WiFi: SSV6501P (driver ssv6x5x)
- Arquitetura: ARMv7l (32-bit)

## Etapas documentadas
- [ ] Identificação do hardware
- [ ] Instalação do Debian
- [ ] Driver WiFi SSV6501P
- [ ] Configuração do Access Point (hostapd + dnsmasq)

## Estrutura
- `configs/` — arquivos de configuração prontos para uso
- `docs/` — documentação detalhada de cada etapa
- `scripts/` — scripts de automação
